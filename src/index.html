Bouncemeet

Bouncemeet is an application made for scheduling, specifically for sports. Bouncemeet automatically provides game archival and generation, sends alerts and emails to specific triggers, and allows users to quickly update their status without even having to log in. Behind the scenes, it's a stack that contains: React, Node.JS, MongoDB, Codeship, Heroku, and Cloudflare.

Pull The Goalie

Ride TUgether

A Python web app made in order to combat the difficulties of life without a car in college. Students can use service the either rides or requests for rides that appear on the map and allow for others to join those rides. Coupled with Facebook's Graph API to gather basic information about a student, the app also integrates with Facebook to allow users to either post rides and notify others.
